# Formação HTML Web Developer - DIO.me

## Sobre o projeto

Este projeto foi desenvolvido como parte da Trilha de HTML da DIO.me, no módulo **Primeiros passos com HTML**.

O objetivo do desafio foi colocar em prática os conceitos iniciais da linguagem HTML, criando uma página simples utilizando as principais tags estudadas durante as aulas.

Foi uma ótima oportunidade para entender melhor a estrutura de uma página web e praticar a escrita de código utilizando apenas HTML.

## Tecnologias utilizadas

- HTML5

## Conteúdos praticados

Durante o desenvolvimento do projeto, foram utilizadas as seguintes tags:

- Títulos (`<h1>` até `<h6>`)
- Parágrafos (`<p>`)
- Destaques de texto (`<mark>`, `<strong>`, `<i>`, `<u>`, `<small>`)
- Listas ordenadas e não ordenadas (`<ol>`, `<ul>`, `<li>`)
- Links (`<a>`)
- Separadores (`<hr>`)
- Texto sobrescrito e subscrito (`<sup>`, `<sub>`)
- Citações (`<blockquote>`)
- Outras tags exploradas durante o desafio, como `<font>`, `<del>` e `<abbr>`

## Aprendizados

Esse desafio marcou meus primeiros passos no desenvolvimento web e me ajudou a compreender como o HTML é responsável pela estrutura e organização do conteúdo de uma página.

Ao longo da formação, a ideia é continuar evoluindo esse conhecimento e aplicar novos conceitos nas próximas etapas do curso.
