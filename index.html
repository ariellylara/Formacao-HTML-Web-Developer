História da Web

(voltar)

A evolução tecnológica começou com os primeiros computadores nas décadas de 1940 e 1950, como o Harvard Mark I, o Colossus e o ENIAC.

A internet surgiu no contexto da Guerra Fria, com a criação da DARPA e estudos sobre redes de comunicação em pacotes.

Em 1969, foi estabelecida a primeira conexão da ARPANet entre a UCLA e o SRI.

O marco fundamental ocorreu em 1989, quando Tim Berners-Lee uniu hipertexto ao protocolo TCP/IP para criar a World Wide Web (WWW), lançando o primeiro navegador em 1991.

O que são Clients?

(voltar)

No ambiente web, os Clients (clientes) são programas, como os navegadores (browsers), utilizados para abrir e executar arquivos como HTML, CSS e Javascript.

Exemplos comuns de clients incluem o Google Chrome, Microsoft Edge, Firefox e Safari.

O cliente é a interface pela qual o usuário acessa a internet, podendo ser um computador, smartphone ou até um smartwatch.

O que são Servers?

(voltar)

Os Servers (servidores) são locais onde os sites e aplicações ficam hospedados.

São compostos por Hardware (processadores, memória e armazenamento) e Software (sistemas operacionais e servidores web).

Existem diversos tipos especializados, como:

  • Web Server: Armazena e entrega o conteúdo dos sites (ex: Apache e Nginx).
  • Database Server: Gerencia bancos de dados.
  • DNS (Domain Name Service): Responsável por localizar o domínio na internet e traduzi-lo em um endereço IP.

Linguagens de programação web

(voltar)

As linguagens de programação websão conjuntos de instruções para gerar software.

Elas se dividem em dois grupos principais:

  • Server-side: Executadas no servidor, como PHP, Java, C#, Ruby e Javascript (via Node.js)
  • Client-side: Executadas diretamente no navegador do usuário, sendo o Javascript o principal exemplo.

Server-side: Executadas no servidor, como PHP, Java, C#, Ruby e Javascript (via Node.js)

Client-side: Executadas diretamente no navegador do usuário, sendo o Javascript o principal exemplo.

NOTA: HTML não é uma linguagem de programação, mas sim uma linguagem de marcação utilizada para estruturar textos que o navegador interpreta.

Ferramentas utilizadas

(voltar)

Para o desenvolvimento web, são citadas ferramentas essenciais como:

  • Inspetor de Elementos: Recurso dos navegadores para analisar e testar o código das páginas
  • Filezilla: Software de conexão FTP para transferir os arquivos do computador local para o servidor de hospedagem
  • Ambiente de desenvolvimento: Configuração necessária para criar e editar os arquivos

Estrutura básica do HTML

(voltar)

A estrutura do HTML é baseada em Tags e Atributos

Os elementos fundamentais incluem:

  • Textos: Organizados por títulos (h1 a h5) e parágrafos (p), com formatações como negrito (strong), itálico (i) e sublinhado (u)
  • Listas: Podem ser ordenadas (ol) ou não ordenadas (ul)
  • Links: Elementos que permitem a navegação entre diferentes páginas ou sites
